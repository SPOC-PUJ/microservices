\doxysubsection{Detailed Description}
This microservice computes the average of multiple complex signals. 

The Signals\+\_\+\+Average microservice is implemented using g\+RPC and Eigen, allowing it to handle complex-\/valued signals and compute their averages efficiently. The service accepts input signals as complex vectors and returns the averaged signal.
