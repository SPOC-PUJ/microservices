\doxysubsection{Detailed Description}
This microservice computes the Fast Wavelet Transform (FWT) of complex signals. 

This module implements a g\+RPC service for computing the Fast Wavelet Transform (FWT) of a signal. It leverages Eigen for matrix operations and convolution.
